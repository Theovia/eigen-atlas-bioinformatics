# Ruta de aprendizaje

## Ruta corta: 1 dia

1. Lee `Fundamentos visuales`.
2. Usa los simuladores de split, PCA, K-means, SVM y red neuronal.
3. Lee `Rigor cientifico`.
4. Resuelve la `Licencia de manejo bioinformatico`.

## Ruta taller: 3 dias

### Dia 1

- Fundamentos de datos biologicos.
- Matrices, etiquetas, split y metricas.
- No supervisado: PCA, clustering y grafos.

### Dia 2

- Supervisado: SVM, arboles, Random Forest.
- CNN para imagenes.
- NLP para textos biologicos.
- Secuencias como k-mers o embeddings.

### Dia 3

- AlphaFold y confianza estructural.
- ChEMBERTa y validacion por scaffold.
- DNNGIOR y auditoria metabolica.
- Model card, bitacora y reproducibilidad.

## Ruta profunda: 2 semanas

- Semana 1: fundamentos, notebooks base y recetas de codigo.
- Semana 2: casos avanzados, papers, referencias oficiales y proyecto final.

Al final, toma un notebook del repo fuente y explica:

```text
entrada -> representacion -> modelo -> salida -> metrica -> limite -> claim
```
